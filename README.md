# Homebrew Tap for nitaiaharoni1 CLI Tools

This tap provides easy installation of CLI tools via Homebrew.

## Installation

```bash
brew tap nitaiaharoni1/tools
```

## Available Formulae

### Gmail CLI

```bash
brew install gmail
```

Command-line interface for Gmail. See [gmail-cli](https://github.com/nitaiaharoni1/gmail-cli) for more information.

### Google Calendar CLI

```bash
brew install google-calendar
```

Command-line interface for Google Calendar. See [google-calendar-cli](https://github.com/nitaiaharoni1/google-calendar-cli) for more information.

### Slack CLI

```bash
brew install slack
```

Powerful command-line interface for Slack built with pure bash. See [slack-cli](https://github.com/nitaiaharoni1/slack-cli) for more information.

**Note:** Installs as `slack-chat` to coexist with official Slack CLI.

### Notion CLI

```bash
brew install notion
```

Powerful command-line interface for Notion built with pure bash. See [notion-cli](https://github.com/nitaiaharoni1/notion-cli) for more information.

## Usage

After installation:

```bash
# Gmail CLI
gmail init
gmail help

# Google Calendar CLI
google-calendar init
google-calendar help

# Slack CLI
slack-chat init
slack-chat help

# Notion CLI
notion init
notion help
```
